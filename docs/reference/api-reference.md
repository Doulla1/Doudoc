# API Reference

## Commands

### `doudoc.openPanel`

Ouvre le panneau principal de lecture.

### `doudoc.refresh`

Rafraîchit l'index documentaire.

## Views

### `doudoc.explorerView`

Affiche la vue compacte dans l'activité bar.
